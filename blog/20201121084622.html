INT16_MIN - int16_tの最小値を表す定数

INT16_MINはint16_t - 符号あり16bit整数型の最小値を表す定数です。「stdint.h」ヘッダをインクルードすることで使えます。C99で導入された定数マクロです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%d", INT16_MIN);
}

printf関数のフォーマット指定子で「%d」を使っています。

出力されたINT16_MINの値です。 

-32768
