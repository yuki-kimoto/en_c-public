INT8_MIN - int8_tの最小値を表す定数

INT8_MINはint8_t - 符号あり8bit整数型の最小値を表す定数です。「stdint.h」をインクルードすると利用できます。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%d", INT8_MIN);
}

INT8_MINの値です。 

-128

INT8_MINはC99で導入されたマクロです。
