Comments

C comments start with "/ *" and end with "* /". Comments have no meaning in the source code. 

/* comment */

You can also write multi-line comments. 

/ *
  Comment 1
  Comment 2
* / /

One-line comment adopted in C99 //

One-line comment "//" was adopted in C99. 

// comment
