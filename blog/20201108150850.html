UINT64_MAX - uint64_tの最大値を表す定数

UINT64_MAXはuint64_t - 符号なし64bit整数型の最大値を表す定数です。「stdint.h」ヘッダをインクルードすると利用できます。C99で導入された定数マクロです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%llu", UINT64_MAX);
}

UINT64_MAXの値です。 

18446744073709551615

UINT64_MAXはC99で導入されたマクロです。printf関数のフォーマット指定子で「%llu」を使っています。
