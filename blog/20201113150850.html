UINT16_MAX - uint16_tの最大値を表す定数

UINT16_MAXはuint16_tの最大値を表す定数です。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%u\n", UINT16_MAX);
}

UINT16_MAXの値です。 

65535

UINT16_MAXはC99で導入されたマクロです。
