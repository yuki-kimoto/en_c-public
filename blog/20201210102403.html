DBL_MAX - doubleの正の最小値を表す定数

DBL_MAXはdouble - 倍精度(64bit)浮動小数点型の正の最大値を表す定数です。「float.h」をインクルードすると利用できます。 

#include <stdio.h>
#include <float.h>

int main(void) {
  printf("%f\n", DBL_MAX);
}

doubleの負の最小値を表す定数

負の最大値はマイナス符号をつけて「-DBL_MAX」で表現できます。
