stdout - 標準出力ストリームを意味するFILE構造体へのポインタ

stdoutは、標準出力ストリームを意味するFILE構造体へのポインタです。標準出力ストリームは、プログラム開始時に自動的にオープンされます。標準出力ストリームは、一般的には、画面への出力を意味していています。stdio.hヘッダを読み込むと使えます。 

#include <stdio.h>

stdout

stdoutは、C言語仕様上は、式として実装されます。stdoutは、グローバル変数あるいは、マクロとして実装可能です。

stdoutを使ったサンプル

stdoutを使ったサンプルです。fprintf関数で、出力先に、標準出力を指定しています。 

#include <stdio.h>

int main(void) {
  const char* message = "Hello";
  fprintf(stdout, "%s\n", message);
}

実行結果です。画面に「Hello」と表示されます。 

Hello
