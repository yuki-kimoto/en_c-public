int16_t - 符号あり16bit整数型

「int16_t」は符号あり16bit整数型です。「stdint.h」ヘッダをインクルードすることで使えます。C99で導入された型です。 

# 符号あり16bit整数型
int16_t

符号あり16bit整数型が表現できる整数の最大値は「32767」、最小値は「-32768」です。

最大値は「INT16_MAX」、最小値は「INT16_MIN」というマクロで定義されています。

int16_tのサンプルコード

int16_tを使ったサンプルコードです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  int16_t num = 1000;
  
  printf("%d\n", num);
}

printf関数のフォーマット指定子で「%d」を使っています。

出力結果。 

1000
