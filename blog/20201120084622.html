uint16_t - 符号なし16bit整数型

「uint16_t」は符号なし16bit整数型です。「stdint.h」をインクルードすることで使えます。C99で導入された型です。 

# 符号なし16bit整数型
uint16_t

符号なし16bit整数型が表現できる整数の最大値は「65535」、最小値は「0」です。

最大値は「UINT16_MAX」というマクロで定義されています。

uint16_tのサンプルコード

uint16_tを使ったサンプルコードです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  uint16_t num = 65535;
  
  printf("%u\n", num);
}

出力結果。 

65535
